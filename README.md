# data-mining-practicum
Dokumentasi praktikum Data Mining (Modul 1–3) meliputi implementasi struktur data Python (List, Tuple, NumPy, Pandas), Exploratory Data Analysis (EDA) seperti cleaning, visualisasi, dan identifikasi pola, serta preprocessing data (missing values, outlier, encoding, normalisasi). Dikerjakan di Google Colab.
